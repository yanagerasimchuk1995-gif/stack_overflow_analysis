# Аналіз даних Stack Overflow  Developer Survey за допомогою Pandas

## Огляд проєкту

Цей проєкт присвячений аналізу результатів опитування розробників Stack Overflow за допомогою бібліотеки Pandas.
У процесі аналізу використовуються методи описової статистики, фільтрації, групування даних та роботи з пропущеними значеннями. Результати аналізу представлені у вигляді таблиць.

---

## Мета проєкту

- Проаналізувати великий реальний датасет розробників
- Дослідити досвід, освіту та поведінку респондентів
- Визначити популярність Python
- Проаналізувати рівень компенсації
- Застосувати Pandas для реального аналітичного дослідження

---

## Інструменти та технології

- Python  
- Pandas  
- Jupyter Notebook

---

## Дані

Проєкт використовує два файли з Stack Overflow Developer Survey:
https://survey.stackoverflow.co/
### survey_results_public.csv
Основний датасет із відповідями респондентів, що містить інформацію про:
- досвід роботи
- освіту
- мови програмування
- компенсацію
- формат роботи

### survey_results_schema.csv
Опис структури датасету (назви та значення колонок)

- Python є однією з найпопулярніших мов програмування серед респондентів
- Значна частина розробників має нерівномірний розподіл досвіду роботи
- Віддалена робота широко поширена серед IT-спеціалістів
- Рівень компенсації суттєво відрізняється між країнами

## Висновок

Аналіз даних Stack Overflow показав різноманітність досвіду, освіти та кар’єрних шляхів розробників у всьому світі.

Python залишається однією з найпопулярніших технологій, а віддалена робота та онлайн-навчання відіграють важливу роль у сучасній IT-індустрії.

Отримані результати дозволяють краще зрозуміти структуру IT-ринку та фактори, що впливають на рівень компенсації та професійний розвиток.
